    🚀 Guide de Configuration Supabase

    📋 Étapes pour activer l'édition complète :

    1. Créer un compte Supabase

    • Allez sur supabase.com
    • Créez un nouveau projet
    • Notez votre URL et clé API

    2. Configurer les tables

    • Allez dans l'éditeur SQL de Supabase
    • Copiez/collez les scripts SQL du code source
    • Exécutez les requêtes pour créer les tables

    3. Ajouter votre premier utilisateur admin

    • Connectez-vous avec votre email
    • Ajoutez-vous comme admin dans la table user_roles
    • Rechargez la page

    4. Test de connexion

    • Utilisez le bouton "Se connecter" dans le header
    • Vérifiez que vous recevez le lien magique par email
    • Une fois connecté, l'interface d'édition sera disponible
